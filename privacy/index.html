Privacy Policy

本网站为静态站点，不主动收集任何个人身份信息。我们不使用 Cookie 追踪用户行为，不运行任何分析脚本，不收集访问日志。

本网站托管于 GitHub Pages，GitHub 可能会根据其自身的隐私政策收集访问者信息。详情请参阅 GitHub 隐私声明 。

第三方服务

联系方式
