About

卫星实验室（XLabs Club）是一个中文开源技术社区，slogan 是「用开源探索边界，用分享传递价值」。我们围绕平台工程（Platform Engineering）、**云原生（Cloud Native）**与研发效能，持续输出工程实践与技术分享，希望以技术驱动业务长期发展与高速增长。

本站（xlabs.club ）为卫星实验室主页源码，合作交流请到我们的 GitHub 主页，欢迎提交 PR、Issue 与 Discussion 共建。

关注方向

  • 平台工程：我们的平台工程建设之路，覆盖 DevOps、DataOps、FinOps 以及 AIOps 的工程实践。
  • 云原生：云原生技术探索，如何以云原生技术支撑起不断变化的复杂业务。
  • 技术博客：研发踩坑记录，翻一翻总有惊喜。

常见问题

  • 卫星实验室是什么？ 卫星实验室（XLabs Club）是一个聚焦平台工程、云原生与 DevOps/AIOps 的中文开源技术社区，通过开源项目与技术博客分享工程实践经验。
  • 如何参与共建？ 在 GitHub 仓库提交 PR、Issue 或 Discussion 即可，我们没有专业运营，GitHub 是主要阵地。
  • 内容采用什么协议？ 文档采用 CC BY-NC 4.0 许可协议，代码遵循各仓库声明的开源协议。
