在建设本博客的过程中，我们发现了一些优秀的开发工具、有趣原创的独立博客，以及独立发布的良心开源产品，觉得有必要把他们展示出来，也算是对创作者的一点回馈。

如有侵权请提 Issue，如有推荐请到我们的 [GitHub](https://github.com/xlabs-club/xlabs-club.github.io) 仓库提交 Pull Request，本页会持续更新。

## 在线工具

- [IT Tools](https://it-tools.tech)：研发人员常用的 JSON / 时间戳 / 数学 / 加密等各类在线生成与转换工具，开源、自托管友好。
- [JS Toolset](https://www.jstoolset.com)：JWT / JSON / URL / Base64 等编解码工具，界面简洁，开箱即用。

## 独立博客

- [地瓜哥的博客](https://www.diguage.com)：分享技术带来的喜悦，内容扎实，更新稳定。

## 独立产品

- 持续收录中，欢迎在 GitHub 仓库提交 Issue 推荐。

如果你也在写独立博客或做开源产品，欢迎通过 PR 把你的作品加入这里，让我们一起让优质内容被更多人看到。
