云原生专题围绕 Kubernetes 生态展开，涵盖弹性伸缩、应用交付、容器化与工程实践等方向，帮助团队系统掌握核心方案。

## 专题内容

- **KEDA 弹性伸缩**：基于事件驱动的 Kubernetes 工作负载自动扩缩容方案，支持 Kafka、Prometheus、RabbitMQ 等多种事件源。
- **容器镜像最佳实践**：Dockerfile 编写、多架构编译、镜像瘦身与安全扫描，以及 ORAS、skopeo 等辅助工具使用。
- **StatefulSet 磁盘扩容**：PV/PVC 平滑扩容的实战方案与各 StorageClass 兼容性分析。
- **cert-manager 证书管理**：使用 Pulumi 部署 cert-manager，为内网环境自动签发和信任自签名证书。
- **长连接负载均衡**：K8S Service 长连接导致流量不均的问题分析与多种解决方案。
